nltk
====

Brings Python [Natural Language Toolkit](https://www.nltk.org/) to R.

------------

``` r
# install.packages("remotes")
remotes::install_github("news-r/nltk")
```

-------

This is a basic example which shows you how to solve a common problem:

``` r

```
